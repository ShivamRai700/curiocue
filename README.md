# CurioCue 🎬

AI-powered movie discovery platform.

## Features
- 🔍 Dynamic movie search
- 🤖 AI explanations (fun + Hinglish style)
- 🎯 Trending recommendations
- 📺 Where to watch

## Tech Stack
- React + Tailwind
- Node.js + Express
- TMDB API
- Google Gemini AI

## Setup
1. Clone repo
2. Add .env
3. Run backend + frontend

## Future Improvements
- Anime support
- Books integration
- Better AI explanations
